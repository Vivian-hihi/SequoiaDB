+-------------------------------------------------+--------------------------------------------------------------------+
| 表达式                                          | 例子                                                               |
+=================================================+====================================================================+
| 单列运算                                        | select a + 1, b / 2 from foo.bar                                   |
+-------------------------------------------------+--------------------------------------------------------------------+
| 复合运算                                        | select ( a + 1 ) * ( a - 1 ), b + 1 from foo.bar                   |
+-------------------------------------------------+--------------------------------------------------------------------+
| 内层查询                                        | select t.b + 1 from ( select a as b from foo.bar split by a ) as t |
+-------------------------------------------------+--------------------------------------------------------------------+

**Note:** 

 -   单列运算，支持加（+）、减（-）、乘（*）、除（/）、模（%）。
 -   单列运算，一个表达式中只能出现一个的字段。
 -   除法、取模运算，被除数为零时返回结果为null。
 -   对非数值型做算术运算时返回的结果为null。
