db.createCS()

方法描述

db.createCS(name,pageSize)方法是创建一个集合空间到数据库对象中。

参数描述

属性 描述 是否必填
name 集合空间名，同一个数据库对象中，集合空间名必须唯一 是
pageSize 数据页大小，默认为4096B 否

格式

createCS()的定义格式包含name与pageSize两个字段。name的值必须为字符串，pageSize则为32位的int类型

("<集合空间名>",[pageSize])

Note:

name参数的命名不能使空串、含点（.）或者美元符号，且长度不能超过127B

创建集合空间时可以指定数据页大小，指定后不可更改。如果不指定默认为4096B。

示例

创建名为foo的集合空间，不指定数据页大小

db.createCS(“foo”)

创建一个名为foo的集合空间，指定数据页大小为1024B

db.createCS(“foo”,1024)
