SYSCOLLECTIONSPACES集合

所属集合空间

SYSCAT

概念

SYSCOLLECTIONSPACES集合中包含了该集群中所有的用户集合空间信息。每个用户集合空间保存为一个文档。

每个文档包含以下字段：

字段名 类型 描述
Name 字符串 集合空间名
Collection 数组 该集合空间中包含的所有的集合名，每个集合为一个JSON对象，包含“Name”字段与相应的集合名
Group 数组 该集合空间所在的分区组ID
PageSize 整数 该集合空间的数据页大小

示例

一个典型的包含一个集合，存放在一个分区组中的集合空间如下：

{ "Collection" : [ { "Name" : "foo" } ], 
  "Group" : [ { "GroupID" : 1000 } ], 
  "Name" : "test", 
  "PageSize" : 4096 }
