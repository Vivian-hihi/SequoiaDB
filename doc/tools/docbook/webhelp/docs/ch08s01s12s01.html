OID

Json 格式

{ "$oid" : <24字节16进制字符串> }

    函数格式

    格式 描述

    ObjectId()

    生成一个 OID

    ObjectId(24字节16进制字符串)

    生成一个指定值的 OID

      示例

      • 按照 OID 查询 

      > db.foo.bar.find({"_id":{"$oid": "55713f7953e6769804000001"}})
> db.foo.bar.find({_id:ObjectId("55713f7953e6769804000001")})

      • 按照 OID 查询 

      > db.foo.bar.insert({id:ObjectId()})
