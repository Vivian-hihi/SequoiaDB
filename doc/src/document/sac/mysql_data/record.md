**MySQL 实例数据操作** 页面可以进行增删改查以及执行 SQL 操作。  

###执行 SQL
在页面中的 SQL 执行框可输入需要执行的 SQL 语句进行执行。
![执行SQL](sac/data_operation/MySQL/exec_sql.png)

###查询操作

点击页面中的查询按钮，填写查询所需要的条件，根据需求填写后点击确定开始查询。如果需要做复杂的查询操作，请使用 SQL 执行框填写 SQL 进行查询。

![查询](sac/data_operation/MySQL/query.png)

###插入操作

点击页面中的插入按钮，填写需要插入的字段及对应的值，如果需要做复杂的插入操作，请使用 SQL 执行框填写 SQL 进行插入。

![插入](sac/data_operation/MySQL/insert.png)

###更新操作

点击页面中的更新按钮，填写更新的条件，根据需求填写后点击确定开始更新。如果需要做复杂的更新操作，请使用 SQL 执行框填写 SQL 进行更新操作。

![更新](sac/data_operation/MySQL/update.png)

###删除操作

点击页面中的删除按钮，填写删除的条件，根据需求填写后点击确定开始删除。如果需要做复杂的删除操作，请使用 SQL 执行框填写 SQL 进行更新操作。

![更新](sac/data_operation/MySQL/update.png)

> **Note:**  
> 以上的操作均可在弹窗中选择执行模式。  
> 当执行模式为马上执行时，点击确定将会马上执行所做的操作。  
> 执行模式为仅生成SQL时，点击确定会将填写的条件生成 SQL 到页面的 SQL 执行框中。

###查看执行记录
查看执行记录可以查看当前页面所执行过的 SQL 语句，包括使用增删改查弹窗进行的操作。点击语句可粘贴到 SQL 执行框。

![执行记录](sac/data_operation/MySQL/history.png)
