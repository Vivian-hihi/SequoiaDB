存储过程页面可以查看当前服务的所有存储过程信息，并且可以进行创建和删除存储过程操作。

![存储过程](sac/monitor/resource/procedures_1.jpg)

- 该页面显示服务的所有存储过程信息

- 存储过程函数在该页面将被格式化显示。

- 该页面可以进行创建和删除操作。
  - 创建存储过程
      1. 点击表格上方的 **创建存储过程** 按钮打开窗口。
         ![创建存储过程](sac/monitor/resource/procedures_2.jpg)
      2. 在窗口输入框中输入要创建的完整函数，点击确定即可完成创建。

  - 删除存储过程
      1. 在表格中点击要删除的存储过程 **函数名**，打开存储过程详细窗口。
         ![删除存储过程](sac/monitor/resource/procedures_3.jpg)
      2. 点击窗口右下方的 **删除** 按钮即可完成删除。

> **Note:**
>
> - 创建存储过程时请注意：
>   1. 函数必须包含函数名，不能使用如：function(x,y) { return x+y; }。否则将会创建失败。
>   2. 函数中所有标准输出，标准错误会被屏蔽。同时不建议在函数定义或执行时加入输出语句，大量的输出可能会导致存储过程运行失败。
>   3. 函数返回值可以是除 db 以外任意类型数据，如：function getCL() { return db.foo.bar; }。
