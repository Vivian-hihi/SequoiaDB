分区组列表页面可以了解当前服务所有分区组的基本信息和进行启停分区组操作。

![分区组列表](sac/monitor/node/groups_list_1.png)

当需要启停分区组时，可以使用表格上方的 **启动分区组** 、 **停止分区组** 进行操作。

- 启动分区组

  点击 **启动分区组**，选择要启动的分区组，点击 **确定** 按钮。

  ![启动分区组](sac/monitor/node/groups_list_2.png)

- 停止节点

  点击 **停止分区组**，选择要停止的分区组，点击 **确定** 按钮。

  ![停止分区组](sac/monitor/node/groups_list_3.png)

> **Note:**  
> 如果停止编目组，该 SequoiaDB 存储集群将无法使用，需要手工启动编目组。
