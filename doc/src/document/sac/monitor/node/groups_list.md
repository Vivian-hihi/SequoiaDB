分区组列表页面可以了解当前服务所有分区组的基本信息和进行启停分区组操作。

![分区组列表](sac/monitor/node/groups_list_1.jpg)

* 页面左上方显示 **coord** 的分区组状态、节点数和连接数。

* 页面左下方显示 **catalog** 的分区组状态、节点数和主节点

* 页面右侧为数据组的表格，可以了解数据组的状态、节点数、主节点、集合数和记录数。

当需要启停分区组时，可以使用表格上方的 **启动分区组** 、 **停止分区组** 进行操作。

- 启动分区组
  1. 点击上方 **启动分区组** 按钮，打开启动分区组的窗口。
     ![启动分区组](sac/monitor/node/groups_list_2.jpg)
  2. 通过下拉菜单选择要启动的分区组，点击确定。

- 停止节点
  1. 点击上方 **停止分区组** 按钮，打开停止分区组的窗口。
     ![停止分区组](sac/monitor/node/groups_list_3.jpg)
  2. 通过下拉菜单选择要停止的分区组，点击确定。

> **Note:**
>
> 选择停止catalog组后，sac服务将无法使用，需要手工启动catalog组。
