
* 浏览器

  SAC可以使用以下浏览器:
  1. Chrome，支持Chrome 19.0以上版本，建议使用当前较新的版本。
  2. Firefox，支持Firefox 22.0以上版本，建议使用当前较新的版本。
  3. IE7/8/9+，为了更好使用体验， 建议使用IE9或更高版本。
  4. Microsoft Edge，建议使用较新版本。
  5. 其他主流浏览器，SAC支持各种主流浏览器，国内浏览器通常是多内核，在不能正常使用可以尝试切换内核，建议使用较新的版本。

* 分辨率和缩放
  1. SAC页面需要分辨率不小于 1024 * 768，为了更好使用体验，建议分辨率在 1366 * 768或更高。
  2. SAC页面支持缩放，缩放默认是100%，SAC支持80%、90%、100%、125%。

* 浏览器设置要求
  1. IE浏览器可能需要把网站添加到 **可信站点** 才能正常使用。  
  2. 浏览器必须允许网站运行 Javascript（默认允许）。
