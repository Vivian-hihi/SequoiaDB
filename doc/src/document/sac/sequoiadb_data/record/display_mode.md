
演示用的记录：

```lang-json
{
   "name": "Jack",
   "phone": "136123456",
   "address": "Guangzhou City, Guangdong Province"
}
{
   "name": "Mike",
   "phone": "136456789",
   "address": "Guangzhou City, Guangdong Province"
}
```

1. 演示已经在 **foo.bar** 集合插入两条记录，插入记录可以参考 [插入记录文档](sac/sequoiadb_data/record/insert.md)。

   ![显示模式](sac/sequoiadb_data/record/display_mode_1.png)

2. 显示模式默认是字符串模式，在记录表格的底部工具栏有三个显示模式。

   ![显示模式](sac/sequoiadb_data/record/display_mode_1.png)

3. 点击第二个图标，这是 JSON 树状图模式。点击字段可以展开嵌套。

   ![显示模式](sac/sequoiadb_data/record/display_mode_2.png)

3. 点击第三个图标，这是表格模式。

   ![显示模式](sac/sequoiadb_data/record/display_mode_3.png)
