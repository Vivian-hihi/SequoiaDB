
演示用的记录：

```lang-json
{
   "name": "Jack",
   "phone": "136123456",
   "address": "Guangzhou City, Guangdong Province"
}
{
   "name": "Mike",
   "phone": "136456789",
   "address": "Guangzhou City, Guangdong Province"
}
```

1. 演示已经在 foo.bar 的集合插入两条记录，
   插入记录可以参考[插入文档](sac/sequoiadb_data/record/insert.md)。

   ![显示模式](sac/data_operation/record/display_mode_1.jpg)

2. 显示模式默认是字符串模式，可以在记录表格的底部工具栏看到 **A** 图标是点亮的。

   ![显示模式](sac/data_operation/record/display_mode_1.jpg)

3. 点击第二个图标，这是JSON树状图模式。点击记录是可以展开查看的。

   ![显示模式](sac/data_operation/record/display_mode_2.jpg)

3. 点击第三个图标，这是表格模式。

   ![显示模式](sac/data_operation/record/display_mode_3.jpg)
