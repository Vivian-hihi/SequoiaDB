
1. 点击导航 **数据 - 分布式存储** 的名字，进入 **集合** 分页。

   ![集合属性](sac/sequoiadb_data/collection/attribute_1.png)

2. 点击表格的 **集合名**，可以切换显示右边的集合信息。

   ![集合属性](sac/sequoiadb_data/collection/attribute_2.png)

3. 集群模式下，集合有 Group 属性，可以切换分区组查看单个组的信息。

   ![集合属性](sac/sequoiadb_data/collection/attribute_3.png)

4. 集群模式下，集合的分区类型是水平分区或者垂直分区，集合有 Partitions 属性，点击 **显示** 可以查看相关的分区信息。

   ![集合属性](sac/sequoiadb_data/collection/attribute_4.png)
