
1. 进入业务的数据操作主页，点击导航 **数据**，点击 **业务名**，演示中点击的是 **myModule1**，点击 **集合** 分页。

   ![集合属性](sac/data_operation/collection/attribute_1.jpg)

2. 点击表格的集合名，即可切换查看右边的集合信息。

   ![集合属性](sac/data_operation/collection/attribute_2.jpg)

3. 集群模式下，集合有Group属性，可以切换分区组查看单个组的信息。

   ![集合属性](sac/data_operation/collection/attribute_3.jpg)

4. 集群模式下，集合的分区类型是水平分区或者垂直分区，集合有Partitions属性，点击 **显示** 可以查看相关的分区信息。
   - 这是水平散列分区的集合分区信息。

      ![集合属性](sac/data_operation/collection/attribute_4.jpg)
   - 这是垂直分区的集合分区信息。

      ![集合属性](sac/data_operation/collection/attribute_5.jpg)
