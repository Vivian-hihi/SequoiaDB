**PostgreSQL 实例数据操作** 页面可以进行增删改查以及执行SQL操作。  

执行 SQL
---

在 **SQL 执行框**输入 SQL 语句，点击 **执行**。

![执行SQL](sac/postgresql_data/data_1.png)

查询操作
---

点击 **查询** 按钮，填写查询条件，点击 **确定** 开始查询。

> **Note:**  
> 需要做复杂的查询操作，请使用 SQL 执行框填写 SQL 进行查询。

![查询](sac/postgresql_data/data_2.png)

插入操作
---

点击 **插入** 按钮，填写插入参数，点击 **确定** 开始插入。

> **Note:**  
> 需要做复杂的插入操作，请使用 SQL 执行框填写 SQL 进行插入。

![插入](sac/postgresql_data/data_3.png)

更新操作
---

点击 **更新** 按钮，填写更新条件，点击 **确定** 开始更新。

> **Note:**  
> 需要做复杂的更新操作，请使用 SQL 执行框填写 SQL 进行更新操作。

![更新](sac/postgresql_data/data_4.png)

删除操作
---

点击 **删除** 按钮，填写删除条件，点击 **确定** 开始删除。

> **Note:**  
> 需要做复杂的删除操作，请使用 SQL 执行框填写 SQL 进行删除操作。

![删除](sac/postgresql_data/data_5.png)

> **Note:**  
> 以上的操作均可在弹窗中选择执行模式。  
> 当执行模式为马上执行时，点击确定将会马上执行所做的操作。  
> 执行模式为仅生成SQL时，点击确定会将填写的条件生成 SQL 到页面的 SQL 执行框中。

查看执行记录
---

查看执行记录可以查看当前页面所执行过的 SQL 语句，包括使用增删改查弹窗进行的操作。点击语句可粘贴到 SQL 执行框。

![执行记录](sac/postgresql_data/data_6.png)
