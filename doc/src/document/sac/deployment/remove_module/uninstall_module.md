
1. 进入 **部署** 页面。

   ![卸载服务](sac/deployment/remove_module/uninstall_1.jpg)

2. 点击 **删除服务 - 卸载服务**。

   ![卸载服务](sac/deployment/remove_module/uninstall_2.jpg)

3. 在窗口选择要卸载的服务。  
   **清除所有数据** 默认关闭，如果关闭，SAC 会检查服务是否含有数据，有数据则不会卸载服务并报错；  
   如果开启，SAC 忽略服务数据，直接卸载服务。

   ![卸载服务](sac/deployment/remove_module/uninstall_3.jpg)

4. 等待任务完成。

   ![卸载服务](sac/deployment/remove_module/uninstall_4.jpg)

5. 卸载完成，点击 **完成** 按钮回到 **部署** 页面。

   ![卸载服务](sac/deployment/remove_module/uninstall_5.jpg)
