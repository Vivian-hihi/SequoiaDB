
> **Note:**  
> 卸载SequoiaDB业务必须先把数据都删除，否则卸载会提示集合空间有数据（防止误操作导致数据丢失）。

1. 打开部署首页。

   ![卸载业务](sac/deployment/uninstall_module_1.jpg)

2. 点击 **卸载业务**， 在卸载业务窗口选择要卸载的业务。

   ![卸载业务](sac/deployment/uninstall_module_2.jpg)

3. 等待卸载任务完成。

   ![卸载业务](sac/deployment/uninstall_module_3.jpg)

4. 卸载完成，点击 **完成**。

   ![卸载业务](sac/deployment/uninstall_module_4.jpg)
