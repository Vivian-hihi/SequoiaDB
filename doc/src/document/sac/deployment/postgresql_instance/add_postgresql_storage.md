> **Note：**  
> PostgreSQL 实例可以添加多个存储集群。

1. 进入 **部署 - 数据库实例** 页面。

   ![数据库实例](sac/deployment/postgresql_instance/add_postgresql_storage_1.png)

2. 点击 **配置实例存储 - 添加实例存储**。

   ![数据库实例](sac/deployment/postgresql_instance/add_postgresql_storage_2.png)

3. 配置好参数，点击 **确定** 按钮。

   ![数据库实例](sac/deployment/postgresql_instance/add_postgresql_storage_3.png)

4. 添加成功。

   ![数据库实例](sac/deployment/postgresql_instance/add_postgresql_storage_4.png)

5. **部署** 页面提示这个实例已经有存储集群了。

   ![数据库实例](sac/deployment/postgresql_instance/add_postgresql_storage_5.png)

6. 现在在这个实例可以创建外部表，外部表的数据将会落在存储集群中，[点击查看](sac/postgresql_data/database.md#创建数据表)。
