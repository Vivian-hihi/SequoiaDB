
> **Note:**  
> 删除主机必须先卸载业务，否则卸载会提示主机有业务存在（防止误操作导致卸载业务）。  
> 卸载业务，[点击查看](sac/deployment/uninstall_module.md)

1. 打开部署首页，点击 **主机** 分页。

   ![删除主机](sac/deployment/remove_host_1.jpg)

2. 选择要删除的主机，全部选择可以点击 **全选** 按钮，点击 **删除主机** 按钮。

   ![删除主机](sac/deployment/remove_host_2.jpg)

3. 等待任务完成。

   ![删除主机](sac/deployment/remove_host_3.jpg)

4. 删除完成，点击 **完成**。

   ![删除主机](sac/deployment/remove_host_4.jpg)
