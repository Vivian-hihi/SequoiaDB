解绑主机是将选择的主机从SAC中将该主机分离出来，不会卸载该主机。
> **Note:**  
> 解绑主机必须先卸载或者解绑服务，否则解绑会提示主机有服务存在。  
> 卸载服务，[点击查看](sac/deployment/remove_module/uninstall_module.md)  
> 解绑服务，[点击查看](sac/deployment/remove_module/unbind_module.md)  

1. 打开部署首页，点击主机分页，选择要删除的主机，全部选择可以点击全选按钮，点击删除主机-解绑主机按钮。

   ![解绑主机](sac/deployment/unbind_host_1.jpg)

4. 解绑完成，点击 **完成**。

   ![解绑主机](sac/deployment/unbind_host_2.jpg)
