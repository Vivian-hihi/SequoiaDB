解绑主机是将选择的主机从SAC中将该主机分离出来，不会卸载该主机。
> **Note:**  
> 解绑主机必须先卸载或者解绑服务，否则解绑会提示主机有服务存在。  
> 卸载服务，[点击查看](sac/deployment/remove_module/uninstall_module.md)  
> 解绑服务，[点击查看](sac/deployment/remove_module/unbind_module.md)  

1. 进入 **部署 - 主机** 页面。

   ![解绑主机](sac/deployment/remove_host/unbind_host_1.jpg)

2. 选择要解绑的主机，可以点击 **全选** 按钮选择全部。

   ![解绑主机](sac/deployment/remove_host/unbind_host_2.jpg)

3. 点击 **删除主机-解绑主机** 按钮。

   ![解绑主机](sac/deployment/remove_host/unbind_host_3.jpg)

4. 解绑完成，点击 **确定**。

   ![解绑主机](sac/deployment/remove_host/unbind_host_4.jpg)

5. 演示的三台主机都解绑了。

   ![解绑主机](sac/deployment/remove_host/unbind_host_5.jpg)
