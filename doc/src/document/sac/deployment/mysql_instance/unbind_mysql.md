**移除 MySQL 实例** 是把实例从 SAC 管理中移除；可以通过 **添加已有的实例** 把移除的实例添加回来，[点击查看](sac/deployment/mysql_instance/append_mysql.md)。

1. 进入 **部署 - 数据库实例** 页面。

   ![数据库实例](sac/deployment/mysql_instance/unbind_mysql_1.png)

2. 点击 **删除实例 - 移除实例**。

   ![数据库实例](sac/deployment/mysql_instance/unbind_mysql_2.png)

3. 选择要移除的实例，点击 **确定** 按钮。

   ![数据库实例](sac/deployment/mysql_instance/unbind_mysql_3.png)

4. 移除实例成功。

   ![数据库实例](sac/deployment/mysql_instance/unbind_mysql_4.png)
