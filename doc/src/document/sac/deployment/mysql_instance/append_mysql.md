1. 进入 **部署 - 数据库实例** 页面。

   ![数据库实例](sac/deployment/mysql_instance/append_mysql_1.png)

2. 点击 **添加实例 - 添加已有的实例**。

   ![添加 MySQL 实例](sac/deployment/mysql_instance/append_mysql_2.png)

3. 实例类型选择 **MySQL**，输入参数，点击 **确定** 按钮

   **地址**：必填，MySQL 数据库的主机名或者 IP 地址。  
   **端口**：必填，MySQL 数据库的监听端口号。  
   **数据库用户名**： 必填，MySQL 数据库的用户名。  
   **数据库密码**：选填，MySQL 数据库的密码。

   ![添加 MySQL 实例](sac/deployment/mysql_instance/append_mysql_3.png)

4. 添加实例完成，这里可以查看 MySQL 实例的配置，点击 **返回** 按钮。

  ![添加 MySQL 实例](sac/deployment/mysql_instance/append_mysql_4.png)

5. **部署 - 数据库实例** 页面已经显示出 MySQL 实例。

  ![添加 MySQL 实例](sac/deployment/mysql_instance/append_mysql_5.png)
