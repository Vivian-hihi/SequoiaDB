关联业务是将不同类型的业务建立关联。目前只支持SequoiaSQL-OLTP业务关联SequoiaDB业务。

一、 建立关联

1. 通过SAC创建了SequoiaDB业务和SequoiaSQL-OLTP业务后，点击部署首页的关联业务-创建关联进行选择需要关联的业务及参数填写。
![创建关联](sac/deployment/relation_2.jpg)

2. 创建成功之后即可在面板看到业务之间的关联信息。
![关联信息](sac/deployment/relation_4.jpg)

二、 解除关联

1. 点击部署首页的关联业务-解除关联进行选择需要解除的关联。
![解除关联](sac/deployment/relation_6.jpg)
