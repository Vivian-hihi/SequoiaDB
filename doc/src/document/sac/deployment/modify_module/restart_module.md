1.  在 **部署页面** 点击 **服务操作** 后选择 **重启服务**。

   ![打开重启](sac/deployment/modify_module/restart_1.jpg)

2.  在弹出的窗口中选择需要重启的服务，点击确定。

   ![重启弹窗](sac/deployment/modify_module/restart_2.jpg)

3.  开始重启服务，等待重启完成。

   ![开始重启](sac/deployment/modify_module/restart_3.jpg)

4.  重启完成。

   ![重启完成](sac/deployment/modify_module/restart_4.jpg)
