同步服务可以让修改后的SequoiaDB服务配置更新到om中。

1. 进入 **部署** 页面。

    ![同步服务](sac/deployment/modify_module/sync_1.jpg)

2. 点击 **服务操作 - 同步服务**。

    ![同步服务](sac/deployment/modify_module/sync_2.jpg)

3. 选择要同步的服务。

    ![同步服务](sac/deployment/modify_module/sync_3.jpg)

4. 同步完成，为了展示效果，演示环境修改了全部节点配置。

    ![同步服务](sac/deployment/modify_module/sync_4.jpg)

> **note:**  
> 同步完成后配置有变化的节点会在表格第一列显示修改、删除或新增的标识。
