部署包是将指定的包部署到指定的主机中。目前只支持部署SequoiaPostgreSQL包。

> **Note:**  
> 部署包之前需要将包放在SAC所在主机的安装路径packet路径下，例：/opt/sequoiadb/packet。

1. 从部署首页-主机页面勾选需要部署包的机器，点击主机操作-部署包进入配置页面填写参数。

   ![部署包](sac/deployment/package_3.jpg)

   > **Note:**  
   > 部署时有非强制安装和强制安装两种选项，当选择非强制安装时，点击下一步时不会往已经装有选择的包的主机进行安装。当选择强制安装时，会在选择的每台机器上重新安装，并且会强制重启对应机器的对应业务。

2. 填写好参数之后，点击下一步即可进行安装。

   ![部署包](sac/deployment/package_6.jpg)
