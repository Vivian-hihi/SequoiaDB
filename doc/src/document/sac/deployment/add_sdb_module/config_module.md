
添加业务流程共3步骤：

| 步骤 | 说明 |
| ---- | ------------- |
| [配置业务](sac/deployment/add_sdb_module/config_module.md)  | 配置业务部署模式、主机、节点分布。 |
| [修改业务](sac/deployment/add_sdb_module/mod_module.md)     | 修改业务的配置参数。 |
| [安装业务](sac/deployment/add_sdb_module/install_module.md) | 安装业务的服务。 |

1. 配置业务，根据实际需求配置。

   ![配置业务](sac/deployment/add_sdb_module/config_module_1.jpg)

2. 点击 **选择安装业务的主机**，可以指定主机安装业务，默认全选。

   ![配置业务](sac/deployment/add_sdb_module/config_module_2.jpg)

3. 演示将会使用3台主机安装SequoiaDB集群，3副本，3分区组

   ![配置业务](sac/deployment/add_sdb_module/config_module_3.jpg)

4. 点击底部的 **下一步** 按钮，进入修改业务页面。

5. 开始修改业务，[点击查看](sac/deployment/add_sdb_module/mod_module.md)。
