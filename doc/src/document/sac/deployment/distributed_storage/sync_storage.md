**同步配置** 可以把修改后的存储集群配置同步到 SAC 中。

1. 进入 **部署 - 分布式存储** 页面。

   ![同步配置](sac/deployment/distributed_storage/sync_1.png)

2. 点击 **存储集群操作 - 同步配置**。

   ![同步配置](sac/deployment/distributed_storage/sync_2.png)

3. 选择要同步配置的存储集群。

   ![同步配置](sac/deployment/distributed_storage/sync_3.png)

4. 同步完成，为了展示效果，演示环境修改了全部节点配置。

   > **Note:**  
   > 同步完成，配置有变化的节点会在表格第一列显示修改、删除或新增的标识。

   ![同步服务](sac/deployment/distributed_storage/sync_4.png)
