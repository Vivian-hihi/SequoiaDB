1. 在 **部署页面 - 分布式存储** 页面，点击 **存储集群操作 - 重启**。

   ![打开重启](sac/deployment/distributed_storage/restart_1.png)

2. 选择需要重启的存储集群，点击 **确定** 按钮。

   ![重启弹窗](sac/deployment/distributed_storage/restart_2.png)

3. 等待任务完成。

   ![开始重启](sac/deployment/distributed_storage/restart_3.png)

4. 任务完成。

   ![重启完成](sac/deployment/distributed_storage/restart_4.png)
