
1. 打开浏览器，地址为OM服务的IP，访问端口默认8000。

2. 例如 http://192.168.1.101:8000

3. 登录用户名默认 admin，初始密码 admin

4. 输入默认密码 admin

5. 点击 **登录** 按钮

![登录SAC](sac/install_login/login.jpg)

> **Note:**  
> 登录成功后，可以修改SAC密码，[点击查看](sac/install_login/reset_pwd.md)。  
> 第一次使用SAC，可以通过一键部署来完成集群安装，[点击查看](sac/deployment/deployment_wizard.md)。
