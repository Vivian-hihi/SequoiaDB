###插入(Insert)###
1. 点击 < **插入** > 按钮。

  ![点击插入按钮](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_1.jpg)

2. 插入界面介绍。

  ![界面](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_2.jpg)

3. 通过图形化构建一条记录。 { "name": "Jack", "age": 22, "phone": [ "123", "456" ], "time": { "$date": "2015-01-01" } }   
  
  1. 添加一个新字段，编辑字段名为 **name**，编辑值为 **Jack**。

     ![添加字段name](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_3.jpg)

     ![添加字段name](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_4.jpg)

  2. 添加一个新字段，编辑字段名为 **age**，编辑类型为 **Auto** ，编辑值为 **22**。

     ![添加字段age](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_5.jpg)

  3. 添加一个新字段，编辑字段名为 **phone**，编辑类型为 **Array** 。

     ![添加字段phone](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_6.jpg)

  4. 在 **phone** 添加2个新数组元素，编辑值为 **123** 和 **456**。

     ![编辑字段phone](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_7.jpg)

  5. 添加一个新字段，编辑字段名为 **time**，编辑类型为 **Date** ，编辑值为 **2015-01-01**。

     ![添加字段time](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_8.jpg)

     ![添加字段time](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_9.jpg)

  6. 点击 < **确定** >。

     ![点击确定](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_10.jpg)

4. 除了使用图形化构建记录，也可以切换成字符串模式，输入一条Json也是可以的。

  ![字符串模式构建记录](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_11.jpg)

5. 插入成功。

  ![插入成功](installation/deployment/visualization_data_operation/visualization_data_operation_insert_cn_12.jpg)

###复制(Copy)###

为了方便构建相似或重复的记录，提供 复制 功能来实现。  

1. 点击记录的 < **复制** > 按钮。

  ![点击复制按钮](installation/deployment/visualization_data_operation/visualization_data_operation_copy_cn_1.jpg)

2. 按需求修改记录，点击 < **确定** >。

  ![修改记录](installation/deployment/visualization_data_operation/visualization_data_operation_copy_cn_2.jpg)

3. 复制成功。

  ![复制成功](installation/deployment/visualization_data_operation/visualization_data_operation_copy_cn_3.jpg)

  ![查询成功](installation/deployment/visualization_data_operation/visualization_data_operation_query_cn_4.jpg)
