通过OM部署好数据库之后，就可以进行数据库操作了。

##推荐浏览器##

支持IE7/8/9/10/11, Microsoft Edge, Chrome, Firefox等。

推荐使用Chrome, Microsoft Edge, Firefox。

##安装 OM 服务##

安装OM服务、部署SequoiaDB数据库，请参考 [可视化安装](installation/deployment/visualization_installation.md) 。

###进入数据库操作###

- 在首页点击集群对应的< **业务** >，进入业务列表页面。

  点击 **myCluster** 集群的< **业务** >。

  ![查看集群业务](installation/deployment/visualization_data_operation/visualization_data_operation_cn_1.jpg)

- 点击要进行操作的< **业务名** >;，进入数据操作页面。

  点击 **myModule** 业务。

  ![选择业务](installation/deployment/visualization_data_operation/visualization_data_operation_cn_2.jpg)
    
- 点击< **数据库操作** >
    
  ![点击数据库操作](installation/deployment/visualization_data_operation/visualization_data_operation_cn_11.jpg)

###进入数据操作###

- 在首页点击集群对应的< **业务** >，进入业务列表页面。

  如图，点击 **myCluster** 集群的< **业务** >。

  ![进入业务列表](installation/deployment/visualization_data_operation/visualization_data_operation_cn_1.jpg)

- 点击要进行操作的< **业务名** >，进入数据操作页面。

  如图，点击 **myModule** 业务。

  ![进入数据操作页面](installation/deployment/visualization_data_operation/visualization_data_operation_cn_2.jpg)
    
- 如图，点击 **集合名** 、 **记录数** 都可以进入记录操作页面，点击 **Lob数** 则进入Lob操作页面。
    
  ![进行数据操作](installation/deployment/visualization_data_operation/visualization_data_operation_cn_3.jpg)
