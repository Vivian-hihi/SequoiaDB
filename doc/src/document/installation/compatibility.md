| 3.0.*      | X         | X         | X          | X            | X         | X           | X           | √           |  

- SequoiaDB 不支持版本降级。
- 2.0版本由于lzw算法、以及lob存储结构有调整，与其他版本不兼容，不支持升级到其他版本。
- 对于使用了全文检索的环境，升级过程中要同时升级并重启适配器进程。由于 3.0 之后的版本对全文索引相关机制进行了调整优化，如果升级前版本为 3.0，在升级过程中，需要在停数据节点前，先将所有适配器停止，然后再进行节点升级，或者在升级前将全文索引删除，在升级完成后重建。
