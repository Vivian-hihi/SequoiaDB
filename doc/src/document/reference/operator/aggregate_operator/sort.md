##语法##

```
{ $sort: { 字段1: 0 | 1 [, 字段2: 0 | 1, ... ] } }
```

##描述##

$sort用来指定结果集的排序规则。对嵌套对象使用点操作符（.）引用字段名。

##示例##

```lang-javascript
> db.foo.bar.aggregate({ $sort: { score: -1, name: 1 } })
```

该操作表示从集合foo.bar中读取记录，并以score的字段值进行降序排序（1表示升序，-1表示降序）;

当记录间score字段值相同时，则以name字段值进行升序排序。
