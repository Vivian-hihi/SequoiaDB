##语法##

***System.killProcess( \<optionObj\> )***

##类别##

System

##描述##

杀死指定进程

##参数##

| 参数名    | 参数类型 | 默认值 | 描述         | 是否必填 |
| --------- | -------- | ------ | ------------ | -------- |
| optionObj  | JSON   | ---    | 进程信息     | 是       |

options 参数详细说明如下：

| 属性     | 值类型 | 是否<br>必填 | 格式 | 描述 |
| -------- | ------ | -------- | -------------------- | ---------------------------------- |
| pid    | int |     是   | { pid: 31831 }     | 指定pid                       |

##返回值##

无

##错误##

[错误码](reference/Sequoiadb_error_code.md)

##示例##

* 杀死指定进程：

  ```lang-javascript
  > System.killProcess({pid:31831})
  ```
