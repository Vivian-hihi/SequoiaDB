用于限制返回记录个数。

##语法##
***limit\<limit_num\>***

##参数##
| 参数名 | 参数类型 | 描述 | 是否必填 |
|--------|----------|------|----------|
| limit_num | int | 限制返回的记录数。 | 是 |

##返回值##
无。

##示例##

   * 集合 foo.bar 中原始记录。

   ```
   {a:0, b:2}
   {a:1, b:2}
   {a:2, b:3}
   {a:3, b:3}
   ```

   * 返回集合中前2条记录。

   ```lang-javascript
   > db.exec("select * from foo.bar limit 2")
   { "_id": { "$oid": "58297592fa75cf402400000b" }, "a": 0, "b": 2 }
   { "_id": { "$oid": "58297592fa75cf402400000c" }, "a": 1, "b": 2 }
   Return 2 row(s).
   Takes 0.4708s.
   ```
