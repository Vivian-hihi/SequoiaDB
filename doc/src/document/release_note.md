SequoiaDB 数据库是一款新型企业级分布式非关系型数据库，帮助企业用户降低 IT 成本，并对大数据的存储与分析提供了一个坚实，可靠，高效与灵活的底层平台。

[快速使用SequoiaDB](quickstart.md)

##SequoiaDB version 3.0.1 版本说明##

**接口变更：**

- 增加 snapshot 新增参数，可指定查询条件、过滤条件，对结果进行排序等
- 增加 LOB 判断 eof 的接口

**主要特性：**

- SAC支持 SequoiaSQL-MySQL 与 SequoiaDB 创建关联
- SAC支持 SequoiaSQL-MySQL 添加业务和卸载业务
- 集合空间、集合增加全局唯一 ID

**解决重要Bug：**

- 分区表使用特定条件查询，匹配数据不完整
- split与alter并发时，导致数据组源节点cl在切分清理数据节点被删除
- 插入LOB的同时切分并数据源主节点断网，丢失一个LOB
- 创建全文索引并立即插入大量数据的情况下，主备数据可能出现固定集合中数据不一致
- 主子表中存在全文索引时，无法直接删除主表
- 插入记录包含_id字段及全文索引字段，_id重复时，固定集合中记录被删除，ES端记录被删除
- 未调用init初始化连接池或者调用close关闭连接池后，执行getNormalCoordNum等方法程序coredump
- 增加节点退出保护，防止节点挂死
- 创建CS过程中数据主节点被kill -9，某个数据节点的LSN一直未同步
