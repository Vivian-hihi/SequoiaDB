SdbQuery 类主要用于指定查询条件和操作查询结果，包含的函数如下：

| 名称 | 描述 |
|------|------|
| arrayAccess() | 将结果集保存到数组中并获取指定下标记录 |
| close() | 关闭游标 |
| count() | 查询符合匹配条件的记录条数 |
| current() | 获取当前游标指向的记录 |
| explain() | 获取查询的访问计划 |
| flags() | 按指定的标志位遍历结果集 |
| getQueryMeta() | 获取查询元数据信息 |
| hint() | 按指定的索引遍历结果集 |
| limit() | 控制查询返回的记录条数 |
| next() | 获取当前游标指向的下一条记录 |
| query() | 以下标的方式访问查询结果集 |
| remove() | 删除查询后的结果集 |
| size() | 返回当前游标到最终游标的记录条数 |
| skip() | 指定结果集从哪条记录开始返回 |
| sort() | 对结果集按指定字段排序 |
| toArray() | 以数组的形式返回结果集 |
| update() | 更新查询后的结果集 |
