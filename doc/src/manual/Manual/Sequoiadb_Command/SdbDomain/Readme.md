SdbDomain 类主要用于操作数据域，包含的函数如下：

| 名称 | 描述 |
|------|------|
| addGroups() | 在域中新增复制组 |
| alter() | 修改域的属性 |
| listCollections() | 枚举域中的集合信息 |
| listCollectionSpaces() | 枚举域中的集合空间信息 |
| removeGroups() | 从域中删除复制组 |
| setAttributes() | 修改域的属性 |
| setGroups() | 重设域包含的复制组 |
| listGroups() | 获取指定域中所有的复制组 |
