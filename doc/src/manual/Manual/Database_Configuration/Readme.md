SequoiaDB 巨杉数据库支持数据库参数配置，用户可以根据需要，通过命令行或修改配置文件的方式对参数进行配置，使得 SequoiaDB 的性能更好的适应业务需求。

本章节主要介绍了配置项参数和特殊配置项参数，具体内容如下：

   - [配置项参数][configuration_parameters]

   - 特殊配置项参数

     - [数据文件][dbpath_modify]
     - [索引文件][indexpath_modify]
     - [大对象数据文件][lobpath_modify]
     - [大对象元数据文件][lobmetapath_modify]
     - [同步日志][log_synchronization]
