节点数据同步页面可以检查该服务节点间的数据是否完全一致。

![数据同步](sac/monitor/node/node_sync_1.png)

通过 **节点显示方式** 可以选择 **全部节点** 或者 **数据不同节点** 的查看方式， **数据不同节点** 方式在表格中只显示数据不一致的节点，通过 **CompleteLSN** 字段检查数据的一致性，如下图所示：
![数据同步](sac/monitor/node/node_sync_2.png)

> **Note:**    
> 1. 选择 **数据不同节点** 方式时，如果节点完全一致，表格中将不显示信息。  
> 2. 通过 **刷新间隔** 和 **启动刷新** 按钮，可以选择获取信息的时间间隔和是否重复获取。
