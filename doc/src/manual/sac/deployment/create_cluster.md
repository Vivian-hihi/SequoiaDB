SAC 的 **集群** 是主机资源集群，用于安装分布式存储、数据库实例等服务。

1. 点击右下方， **创建集群** 按钮。

   ![部署](sac/deployment/create_cluster_1.png)

2. 根据实际需求修改参数，点击 **确定** 按钮。

   ![部署](sac/deployment/create_cluster_2.png)

3. 集群创建成功。

   ![部署](sac/deployment/create_cluster_3.png)

4. 现在可以添加主机 [点击查看](sac/deployment/host/add_host.md)。
