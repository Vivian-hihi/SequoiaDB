1. 进入 **部署 - 分布式存储** 页面。

   ![卸载服务](sac/deployment/distributed_storage/uninstall_1.png)

2. 点击 **删除存储集群**。

   ![卸载服务](sac/deployment/distributed_storage/uninstall_2.png)

3. 选择要卸载的存储集群。  
   **清除所有数据** 默认否。选择否，SAC 会检查存储集群是否含有数据，有数据的情况下不会删除存储集群并且弹出提示；  
   如果选择是，SAC 不会检查数据，开始删除存储集群。

   ![卸载服务](sac/deployment/distributed_storage/uninstall_3.png)

4. 等待任务完成。

   ![卸载服务](sac/deployment/distributed_storage/uninstall_4.png)

5. 删除完成。

   ![卸载服务](sac/deployment/distributed_storage/uninstall_5.png)
