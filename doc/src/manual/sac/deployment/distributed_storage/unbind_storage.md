**移除存储集群** 是把存储集群从 SAC 管理中移除；可以通过 **添加已有的存储集群** 把移除的存储集群添加回来，[点击查看](sac/deployment/distributed_storage/add_attach_storage.md)。

1. 进入 **部署 - 分布式存储** 页面。

   ![分布式存储](sac/deployment/distributed_storage/unbind_1.png)

2. 点击 **删除存储集群 - 移除存储集群**。

   ![移除存储集群](sac/deployment/distributed_storage/unbind_2.png)

3. 选择要移除的存储集群，点击 **确定** 按钮。

   ![移除存储集群](sac/deployment/distributed_storage/unbind_3.png)

4. 移除成功，点击 **确定** 按钮。

   ![移除存储集群](sac/deployment/distributed_storage/unbind_4.png)

5. 该存储集群已经不在 SAC 管理中。

   ![移除存储集群](sac/deployment/distributed_storage/unbind_5.png)
