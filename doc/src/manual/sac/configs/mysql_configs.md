**修改 MySQL 实例配置** 页面可以查看节点配置，可以在线修改节点配置。

通过左侧导航 **配置** 选择 MySQL 实例，点击进入配置页面，可以查看节点的常用配置项，以及进行 **修改配置** 、 **重启服务** 操作。

![配置](sac/configs/mysql_config_1.png)

* 多项修改配置  

  点击 **修改配置** 按钮会弹出修改配置项的弹窗，在弹窗中填写需要修改的项，点击确定完成修改。

  ![查看配置](sac/configs/mysql_config_2.png)  

  * 单项修改配置  

    1. 通过表格中的筛选框查找需要修改的配置项。  

       ![修改配置](sac/configs/mysql_config_3.png)  

    2. 点击所选配置项弹出弹窗，填写需要修改的值之后点击确定。

       ![修改配置](sac/configs/mysql_config_4.png)  

    3. 完成修改。

       ![修改配置](sac/configs/mysql_config_5.png)
