
1. 点击导航 **数据 - 分布式存储** 的名字，进入 **集合** 分页。演示环境已经创建集合，并且为 **user.member** 集合创建了索引。

   创建集合可以参考 [创建集合文档](sac/sequoiadb_data/collection/create_cl.md)，创建索引可以参考 [创建索引文档](sac/sequoiadb_data/index/create_index.md)。

   ![删除索引](sac/sequoiadb_data/index/drop_index_1.png)

2. 点击 **删除索引**，集合选 **user.member**，选择要删除的索引，点击 **确定** 按钮。

   ![删除索引](sac/sequoiadb_data/index/drop_index_2.png)

3. 删除完成，集合 **user.member** 的索引数减少到1。

   ![删除索引](sac/sequoiadb_data/index/drop_index_3.png)
