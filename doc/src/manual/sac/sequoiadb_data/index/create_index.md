
1. 点击导航 **数据 - 分布式存储** 的名字，进入 **集合** 分页。

   演示环境已经创建集合，创建集合可以参考 [创建集合文档](sac/sequoiadb_data/collection/create_cl.md)。

   ![创建索引](sac/sequoiadb_data/index/create_index_1.png)

2. 点击 **创建索引**，选择要创建索引的集合，设置创建索引的参数，点击 **确定** 按钮。

   ![创建索引](sac/sequoiadb_data/index/create_index_2.png)

3. 创建完成， **user.member** 的索引数增加到2。

   ![创建索引](sac/sequoiadb_data/index/create_index_3.png)

4. 查看 **user.member** 的索引信息，选择索引名查看指定索引的详细信息。

   ![创建索引](sac/sequoiadb_data/index/create_index_4.png)
