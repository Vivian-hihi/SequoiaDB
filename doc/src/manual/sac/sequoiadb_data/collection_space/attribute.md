
1. 点击导航 **数据 - 分布式存储** 的名字，进入 **集合空间** 分页。

   ![集合空间属性](sac/sequoiadb_data/collection_space/attribute_1.png)

2. 点击表格的集合空间名，可以切换显示右边的集合空间信息。

   ![集合空间属性](sac/sequoiadb_data/collection_space/attribute_2.png)

3. 没有集合的集合空间，属性是空的。

   ![集合空间属性](sac/sequoiadb_data/collection_space/attribute_3.png)

4. 集群模式下，集合空间有 Group 属性，可以切换分区组查看单个组的信息。

   ![集合空间属性](sac/sequoiadb_data/collection_space/attribute_4.png)
