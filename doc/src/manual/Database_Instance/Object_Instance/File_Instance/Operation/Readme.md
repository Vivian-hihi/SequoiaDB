SequoiaFS 能够将本地目录挂载到 SequoiaDB 巨杉数据库的目标集合，让用户可以使用文件系统 API 对目录进行操作。

本章主要介绍目录操作：

- [挂载目录][mount]
- [卸载目录][unmount]
