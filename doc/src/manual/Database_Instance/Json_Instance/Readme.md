SequoiaDB 巨杉数据库为用户提供了 JSON 实例，通过此实例可以与 SequoiaDB 巨杉数据库的分布式引擎进行交互执行。JSON 实例适用基于 JSON 数据类型的联机业务场景，与 MongoDB 保持部分兼容。用户可以使用 JSON 实例对数据库执行集群管理、运行实例检查、数据增删改查等操作。
