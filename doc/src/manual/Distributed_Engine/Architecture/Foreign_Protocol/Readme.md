SequoiaDB 巨杉数据库提供存储计算分离云服务架构，并支持在此架构上构建多种数据库引擎。客户可以灵活选择需要的数据库引擎，实现无须代码修改的平滑迁移。

目前，SequoiaDB 提供对 MongoDB 的语法兼容，提供完整的事务 ACID 能力，支持 RC 级别 (READ COMMITTED) 事务隔离级别，具备万亿级数据量事务处理能力，满足海量数据的联机业务需求。
