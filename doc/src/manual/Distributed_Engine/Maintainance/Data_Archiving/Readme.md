
为了方便对存量的 SequoiaDB 历史数据进行归档，SequoiaDB 巨杉数据库提供了 sdb-schedule 工具，用户可以借助该工具来实现。sdb-schedule 是一个用于 SequoiaDB 集群间的数据迁移与切换的工具，提供集群间集合迁移和数据切换的能力，能用实现对一个 SequoiaDB 集群上的集合进行定期迁移到另一个作为数据源的 SequoiaDB 集群上的操作。并且利用了 SequoiaDB 的数据源能力，将迁移后的集合进行数据源关联，使得在源集群上能够访问数据源集群上的数据，从而实现数据切换的能力。

该工具适用于数据冷热分离、历史数据归档等场景。 主要能力包括：

- SequoiaDB 集群间集合数据迁移

- 集合数据源关联实现数据切换

- 基于 cron 表达式的周期调度

- Web 可视化管理与运维工具支持
