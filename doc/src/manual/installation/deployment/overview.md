目前，安装部署 SequoiaDB 有两种方式：

* 命令行安装（手工安装）。
* 可视化安装（自动安装）。

命令行安装是通过 SequoiaDB Shell 运行 Javascript 命令，一步一步将环境搭建。使用命令行安装要求用户要对 SequoiaDB 集群的组成及运行机制有比较深入的认识，一般不适合刚开始接触 SequoiaDB 的用户。

可视化安装是在 web 端，通过引导页面一步一步地引导用户将环境搭建。目前，可视化安装可以给用户带来更加便利的安装部署方式。此外，使用可视化安装后，用户还可以在 web 端实时操作、监控和管理整个集群。所以强烈建议用户使用可视化安装的方式，来完成 SequoiaDB 的安装部署。
