##NAME##

sleep - Sleep a few milliseconds.

##SYNOPSIS##

**sleep(\<ms\>)**

##CATEGORY##

Global

##DESCRIPTION##

Sleep a few milliseconds.

##PARAMETERS##

* `ms` ( *Int32*， *Required* )

	Time for sleep, Unit：milliseconds.

##RETURN VALUE##

NULL.

##ERRORS##

NULL.

##HISTORY##

Since v1.0.

##EXAMPLES##

1. Sleep 1 second.

	```lang-javascript
	> sleep(1000)
 	```
