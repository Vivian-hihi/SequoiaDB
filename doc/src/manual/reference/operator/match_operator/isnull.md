##语法##

```lang-json
{ <字段名>: { $isnull: <0|1> } }
```

##描述##

选择集合中指定的“<字段名>”是否为空，或不存在。

| 值  | 作用 |
| --- | ---- |
| 0   | 返回字段存在且不为null的记录。 |
| 1   | 返回字段不存在或为null的记录。 |

## 示例##

* 查询集合 foo.bar 中“age”字段不为空且存在的记录：

  ```lang-javascript
  > db.foo.bar.find( { age: { $isnull: 0 } } )
  ```
