db.removeRG()

NAME

removeRG - Delete the database specified ReplicaGroup.

SYNOPSIS

db.removeRG(<name>)

CATEGORY

Sequoiadb

DESCRIPTION

Delete the database specified ReplicaGroup.

name (string)
ReplicaGroup name, the same database object, a unique ReplicaGroup name.
Format
Definition Format removeRG() method, only the name field, name value is a string, required.
(<"ReplicaGroup name">)
Note:
  • ReplicaGroup name must exist.

Examples

Delete ReplicaGroup called "group" of.

db.removeRG("group")
