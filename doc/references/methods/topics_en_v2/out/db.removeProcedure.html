db.removeProcedure()

NAME

removeProcedure - Deletes the specified function name.

SYNOPSIS

db.removeProcedure(function name)

CATEGORY

Sequoiadb

DESCRIPTION

Deletes the specified function name, function name must exist, otherwise an exception message.

function name (string)
function name.

Defined removeProcedure() method, only one string type parameter name function name, its value must already exits, otherwise an exception.

Examples

Delete the sum function: 

db.removeProcedure("sum")
Function must be guaranteed and consistent definition of the function name. Such as db.removeProcedure('sum') call will return a failure.
