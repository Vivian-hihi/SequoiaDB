db.resetSnapshot()

NAME

resetSnapshot - Reset snapshot.

SYNOPSIS

resetSnapshot([con])

CATEGORY

Sequoiadb

DESCRIPTION

Reset snapshot.

con (json object)
Selecting condition. It merely resets snapshot records that match the conditions in "con". If it is null, it will reset all the snapshot records.
Format
The method "resetSnapshot()" contains the parameter "con". It is a json object.
{["con":"{"Field name 1":{"match 1":"value 1"},"Field name 2":{"match 2":"value 2"}...}"]}

Examples

Reset snapshots that contain the value of "SessionID" greater than 1. 

db.resetSnapshot({SessionID:{$gt:1}})
