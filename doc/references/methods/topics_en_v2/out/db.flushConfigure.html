db.flushConfigure()

NAME

flushConfigure - Brush disk configuration to configuration files.

SYNOPSIS

db.flushConfigure(<rule>)

CATEGORY

Sequoiadb

DESCRIPTION

Brush disk configuration to configuration files.

rule (json object)
Brush rules.
Global
True indicates whether the brush plate system-wide configuration, false indicates that only brush plate node configuration.
Global:true

Examples

Brush database configuration.

db.flushConfigure({Global:true})
SequoiaDB Version 1.8B
