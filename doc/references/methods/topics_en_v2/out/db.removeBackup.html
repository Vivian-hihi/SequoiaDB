db.removeBackup()

NAME

removeBackup - Delete the database backup.

SYNOPSIS

db.removeBackup([options])

CATEGORY

Sequoiadb

DESCRIPTION

Delete the database backup.

options (json object)
Set the backup name, specify the replication group, the backup path and other parameters.
GroupID
Backup copy group ID specified, the default is all replication groups.
GroupID:1000 or GroupID:[1000, 1001]
GroupName
Backup copy group name is specified, the default is all replication groups.
GroupName:"data1" or GroupName:["data1", "data2"]
Name
Backup name, the default view all backups.
Name:"backup-2014-1-1"
Path
Backup path, the default configuration parameters for the specified backup path. The path supports wildcards(g%: group name, h%: host name, s%:service name)
Path:"/opt/sequoiadb/backup/%g"
IsSubDir
The above path "Path" configuration parameters are specified for the backup path configuration parameters subdirectory, the default is false.
IsSubDir:false
Prefix
Backup prefix, supports wildcards(g%,h%,s%), default is empty.
Prefix:"%g_bk_"

Examples

Backup information called "backup-2014-1-1" delete the database.

db.removeBackup({Name:"backup-2014-1-1"})
