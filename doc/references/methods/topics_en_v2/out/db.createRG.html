db.createRG()

NAME

createRG - Create a ReplicaGroup.

SYNOPSIS

db.createRG(<name>)

CATEGORY

Sequoiadb

DESCRIPTION

Create a ReplicaGroup. When creating a ReplicaGroup, the system will automatically generate a "groupID" for a ReplicaGroup.

name (string)
ReplicaGroup name. All ReplicaGroup names are unique to each other in a database object.
Format
The method "createRG()" contains the parameter "name", which is in the type od string. It should not be null.
(<"ReplicaGroup name">)
Note:
  • ReplicaGroup is not null string. It should not contains "." or "$". The length of it should not be greater than 127B.

Examples

Create a ReplicaGroup named "group".

db.createRG("group")
