db.setSessionAttr()

NAME

setSessionAttr - Set Session Properties.

SYNOPSIS

db.setSessionAttr(options)

CATEGORY

Sequoiadb

DESCRIPTION

Set Session Properties.

options (json object)
Session Properties option.
PreferedInstance
Read database instance conversation priority selected logo; value "m"/"M"/"s"/"S"/"a"/"A"/1-7, Respectively master/slave/anyone/node1-node7.
PreferedInstance:"M"

Examples

Set the session priority access to the data from the "master" database instance.

db.setSessionAttr({PreferedInstance:"M"})
SequoiaDB Version 1.6
