db.backupOffline()

NAME

backupOffline - Backup Database.

SYNOPSIS

db.backupOffline([options])

CATEGORY

Sequoiadb

DESCRIPTION

Backup Database.

options (json object)
Set the backup name, specify the replication group, backup and other parameters.
GroupID
Backup copy group ID specified, the default is all replication groups.
GroupID:1000 or GroupID:[1000, 1001]
GroupName
Backup copy group name is specified, the default is all replication groups.
GroupName:"data1" or GroupName:["data1", "data2"]
Name
Backup name, default "YYYY-MM-DD-HH:mm:SS" Backup name time format.
Name:"backup-2014-1-1"
Path
Backup path, the default configuration parameters for the specified backup path.The path supports wildcards (g%: group name, h%: host name, s%:service name).
Path:"/opt/sequoiadb/backup/%g"
IsSubDir
The above Path configuration parameter configuration parameters are specified for the backup path subdircetory, the default is false.
IsSubDir:false
Prefix
Backup prefix, support wildcards (g%,h%,s%), The default is empty.
Prefix:"%g_bk_"
EnableDataDir
Date subdirectories feature is turned on, if the current date is automatically rurned on create a subdirectory "YYYY-MM-DD", the default according to false
EnableDataDir:false
Description
Backup description
Description:"First backup"
EnsureInc
Whether to open an incremental backup, the default is false.
EnsureInc:false
OverWrite
Backup is covered with the same name exists, the default is false.
OverWrite:false

Examples

The entire database full backup.

db.backupOffline({Name:"FullBackup1"})
