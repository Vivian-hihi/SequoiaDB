SequoiaDB javascript方法

db.createCS()

db.createCS(<name>,[pageSize])

在数据库对象中创建集合空间。

参数描述

参数名 参数类型 描述 是否必填
name string 集合空间名。同一个数据库对象中，集合空间名必须唯一。 是
pageSize int(32) 数据页大小。默认为4096B。 否

格式

创建集合空间的定义格式有name和pageSize两个参数，name字段的值为字符串型，pageSize为32的int类型。 

("<集合空间名>",[pagesize])
Note:
  • name字段的值不能是空串、含点（.）或者美元符号（$）。且长度不超过127B。
  • 同一个数据库对象集合空间名必须唯一。
  • 在创建集合空间时用户可以指定数据页大小，指定后不可更改。如果不指定默认为 4096B 。

示例

  • 创建名为foo的集合空间，不指定数据页大小

    db.createCS("foo")

  • 创建名为foo的集合空间，指定数据页大小为1024B

    db.createCS("foo",1024)
