db.getRG()

db.getRG(参数)

返回分片组的引用。

参数描述

参数名 参数类型 描述 是否必填
name string 分片组名。同一个数据库对象中，分片组名唯一 name和id任选
id int 分片组id。创建分组时系统自动分配 name和id任选

格式

getRG()方法定于格式包含name或id字段，name为字符串型，id为int型。 指定的分组名或 id 值要在数据库对象中存在，否则出现操作异常。 

("<分片组名>"|<id>)
Note:
  • name字段的值不能使空串、含点（.），或者美元符号（$）。且长度不超过127B。

示例

  • 指定name值，返回分片组group的引用

    db.getRG("group")

  • 指定id值，返回分片组group的引用

    db.getRG("1000")
