启动SequoiaDB web服务器

通过启动SequoiaDB后台的Web服务，也可以对数据库做各种操作，并且方便管理。

  • 进入/opt/sequoiadb/bin目录
  • 执行命令：
    ./sdbwsart -S <server name:port>

    server name ：指定服务器IP地址，如 192.168.10.10

    port ：指定服务器端口号，如 8080

    最后在浏览器中输入http://192.168.10.10:8080 ，即可访问数据库管理页。
