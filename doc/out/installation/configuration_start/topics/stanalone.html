独立模式的配置与启动

说明：

  （1）本节按照最简部署为例，介绍配置和启动步骤；

  （2）以下操作步骤假设 SequoiaDB 程序安装在 /opt/sequoiadb 目录下；

  （3）sdb 服务进程全部以 sdbadmin 用户运行，请确保所有数据库目录都赋予 sdbadmin 读写权限。

  • 切换到 sdbadmin 用户
    su sdbadmin
  • 进入安装目录下（下文以默认安装路径 /opt/sequoiadb 为例）
    cd /opt/sequoiadb
  • 创建配置文件存放目录
    mkdir -p /opt/sequoiadb/conf/local/11810
    Note: 其中11810为数据库服务端口名，为避免出现端口冲突等问题，切勿将数据库端口配置在随机端口范围以内。如：多数 linux 默认随机端口范围为32768～61000，可将数据库端口配置在32767以下。
  • 拷贝独立模式的样例配置
    cp /opt/sequoiadb/conf/samples/sdb.conf.standalone /opt/sequoiadb/conf/local/11810/sdb.conf
  • 打开配置
    vi /opt/sequoiadb/conf/local/11810/sdb.conf
  • 修改必要的配置参数
    # database path
dbpath=/opt/sequoiadb/database/standalone
    数据库存放路径，根据需要进行配置，并确保路径已经存在（不存在请手工创建）

  • 按 :wq ，保存退出 vi

  • 创建数据文件存放路径
    mkdir -p /opt/sequoiadb/database/standalone
    路径为上一步骤配置的路径
  • 启动数据库进程
    /opt/sequoiadb/bin/sdbstart -c /opt/sequoiadb/conf/local/11810/
  • 数据库配置启动完成
