使用shell的窍门

sequoiadb shell本身内置了帮助文档，通过help命令可以查看。 
> help
function () {
    println("   var db = new Sdb('localhost', 11810)   connect to database");
    println("   db.help()                              help on db methods");
    println("   db.cs.help()                           help on collection space                                                                                                                      cs");
    println("   db.cs.cl                               access collection cl on c                                                                                                                     ollection space cs");
    println("   db.cs.cl.help()                        help on collection cl");
    println("   db.cs.cl.find()                        list all records");
    println("   db.cs.cl.find({a:1})                   list records where a=1");
    println("   print(x), println(x)                   print out x");
    println("   getErr(ret)                        print error description for r                                                                                                                     eturn code");
    println("   quit                                   exit");
    }
  • Database Help

    查看数据库级别帮助使用命令：

    db.help()
  • CollectionSpace Help
    查看集合空间下的方法帮助使用命令：
    db.<collectionspace>.help()
  • Collection Help
    查看集合下的方法帮助使用命令：
    db.<collectionspace>.<collection>.help()
  • Cursor Help

    在shell命令中，当使用find()方法执行数据库读操作时，可是使用各种游标方法和javascript方法来处理find()方法返回的结果集。

    查看游标处理方法帮助使用命令：
    db.<collectionspace>.<collection>.find().help()
  • Methods Help

    此外还有一个了解函数功能的技巧，就是在输入的时候不要输入括号，这样就会显示该函数的javascript源代码。如：

    > db.foo.bar.find
function (query, select) {
    return new SdbQuery(this, query, select);
    }
