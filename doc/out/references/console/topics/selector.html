选择符

为方便理解，给出原始数据样例： 
{
   _id:1,
   class:1,
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 }
           ]
}

$include

说明

选择(!=0)或移除(=0)某个字段

示例

  • db.foo.bar.find({class:1},{students:{$include:1}})

    返回 student 字段

    {
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 },
           ]
}
  • db.foo.bar.find({class:1},{_id:{$include:0}})

    返回移除 _id 字段

    {
   class:1,
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 },
           ]
}

$default

说明

选择某个字段。当字段不存在时返回默认值。可简写为{<fieldName>:<defaultValue>}。

示例

  • db.foo.bar.find({class:1}, {students:[], teacher:{$default:"Mr Liu"}})

    返回

    {
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 },
           ],
   teacher:"Mr Liu"
}

$elemMatch

说明

返回数组内满足条件的元素的集合

示例

  • db.foo.bar.find({class:1}, {students:{$elemMatch:{age:18}}})

    返回

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 },
                 {name:"WangErmazi", age:18 },
           ]
}

$elemMatchOne

说明

返回数组内满足条件的第一个元素的集合

示例

  • db.foo.bar.find({class:1}, {students:{$elemMatchOne:{age:18}}})

    返回

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 }
           ]
}

$slice

说明

返回数组的切片

示例

  • db.foo.bar.find({class:1}, {students:{$slice:1}})

    返回下标为0，长度为1的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:2}})

    返回下标为0，长度为2的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 }，
                 {name:"LiSi", age:19 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:-2}})

    返回倒数前2个元素的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"LiSi", age:19 }，
                 {name:"WangErmazi", age:18 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:[1,2]}})

    返回下标为1，长度为2的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"LiSi", age:19 }，
                 {name:"WangErmazi", age:18 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:[-1,2]}})

    返回倒数第1个开始，长度为2的切片（长度不足，只返回1个元素）

    {
   _id:1,
   class:1,
   students:[
                 {name:"WangErmazi", age:18 }
           ]
}

$abs

说明

返回数字的绝对值，非数字类型返回 null。

示例

  • db.foo.bar.find({},{a:{$abs:1}})
Note: {$abs:1}中1没有特殊含义，仅作为占位符出现。

$ceiling

说明

返回大于目标字段值的最小整数值，非数字类型返回 null。

示例

  • db.foo.bar.find({},{a:{$ceiling:1}})
Note: {$ceiling:1}中1没有特殊含义，仅作为占位符出现。

$floor

说明

返回小于目标字段值的最大整数值，非数字类型返回 null。

示例

  • db.foo.bar.find({},{a:{$floor:1}})
Note: {$floor:1}中1没有特殊含义，仅作为占位符出现。

$mod

说明

返回取模的结果，非数字类型返回 null。

示例

  • 返回字段a对10取模的结果

    db.foo.bar.find({},{a:{$mod:10}})
Note: 不能对零取模。

$add

说明

返回字段值加上某个数值的结果。非数字类型返回 null。

示例

  • 返回a字段加上10的结果。

    db.foo.bar.find({}, {a:{$add:10}})

$subtract

说明

返回字段值减去某个数值的结果。非数字类型返回 null。

示例

  • 返回a字段减去10的结果。

    db.foo.bar.find({}, {a:{$subtract:10}})

$multiply

说明

返回字段值与某个数值相乘的结果。非数字类型返回 null。

示例

  • 返回a字段乘以10的结果。

    db.foo.bar.find({}, {a:{$multiply:10}})

$divide

说明

返回字段值除以某个数值的结果。非数字类型返回 null。

示例

  • 返回a字段除以10的结果。

    db.foo.bar.find({}, {a:{$divide:10}})
Note: 除数不能为0。

$substr

说明

返回字符串的子串，非字符串类型返回 null。

示例

  • 记录{a:"abcdefg"}

    返回下标为0，长度为2的子串。

    db.foo.bar.find({}, {a:{$substr:2}})

    返回{a:" ab"}

    返回倒数第二个字符开始的子串。

    db.foo.bar.find({}, {a:{$substr:-2}})

    返回{a:"fg"}

    返回下标为0，长度为3的子串。

    db.foo.bar.find({}, {a:{$substr:[2,3]}})

    返回{a:"cde"}

    返回倒数第二个字符开始，长度为3的子串。

    db.foo.bar.find({}, {a:{$substr:[-2, 3]}})

    返回{a:"fg"} (长度不足3)

$strlen

说明

返回字符串长度，不包括终止符。非字符串返回 null。

示例

  • db.foo.bar.find({},{a:{$strlen:1})
Note: {$strlen:1}中1没有特殊含义，仅作为占位符出现。

$lower

说明

返回字符串中字符改变为小写的结果。非字符串类型返回 null。

示例

  • db.foo.bar.find({},{a:{$lower:1}})
Note: {a:{$lower:1}}中的1作为占位符出现。

$upper

说明

返回字符串中字符改变为大写的结果。非字符串类型返回 null。

示例

  • db.foo.bar.find({},{a:{$upper:1}})
Note: {a:{$ upper:1}}中的1作为占位符出现。

$ltrim

说明

去掉字符串左侧开头的空格(或制表符)。非字符串类型返回 null。

示例

  • db.foo.bar.find({},{a:{$ltrim:1}})
Note: {a:{$ ltrim:1}}中的1作为占位符出现。

$rtrim

说明

去掉字符串右侧开头的空格(及制表符)。非字符串类型返回 null。

示例

  • db.foo.bar.find({},{a:{$rtrim:1}})
Note: {a:{$ rtrim:1}}中的1作为占位符出现。

$trim

说明

去掉字符串两侧开头的空格(及制表符)。非字符串类型返回 null。

示例

  • db.foo.bar.find({},{a:{$trim:1}})
Note: {a:{$ trim:1}}中的1作为占位符出现。
