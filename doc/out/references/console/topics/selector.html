选择符

为方便理解，给出原始数据样例： 
{
   _id:1,
   class:1,
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 }
           ]
}

$include

说明

选择(!=0)或移除(=0)某个字段

示例

  • db.foo.bar.find({class:1},{students:{$include:1}})

    返回 student 字段

    {
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 },
           ]
}
  • db.foo.bar.find({class:1},{_id:{$include:0}})

    返回移除 _id 字段

    {
   class:1,
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 },
           ]
}

$default

说明

选择某个字段。当字段不存在时返回默认值。可简写为{<fieldName>:<defaultValue>}。

示例

  • db.foo.bar.find({class:1}, {students:[], teacher:{$default:"Mr Liu"}})

    返回

    {
   students:[
              {name:"ZhangSan", age:18 },
              {name:"LiSi", age:19 },
              {name:"WangErmazi", age:18 },
           ],
   teacher:"Mr Liu"
}

$elemMatch

说明

返回数组内满足条件的元素的集合

示例

  • db.foo.bar.find({class:1}, {students:{$elemMatch:{age:18}}})

    返回

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 },
                 {name:"WangErmazi", age:18 },
           ]
}

$elemMatchOne

说明

返回数组内满足条件的第一个元素的集合

示例

  • db.foo.bar.find({class:1}, {students:{$elemMatchOne:{age:18}}})

    返回

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 }
           ]
}

$slice

说明

返回数组的切片

示例

  • db.foo.bar.find({class:1}, {students:{$slice:1}})

    返回下标为0，长度为1的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:2}})

    返回下标为0，长度为2的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"ZhangSan", age:18 }，
                 {name:"LiSi", age:19 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:-2}})

    返回倒数前2个元素的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"LiSi", age:19 }，
                 {name:"WangErmazi", age:18 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:[1,2]}})

    返回下标为1，长度为2的切片

    {
   _id:1,
   class:1,
   students:[
                 {name:"LiSi", age:19 }，
                 {name:"WangErmazi", age:18 }
           ]
}
  • db.foo.bar.find({class:1}, {students:{$slice:[-1,2]}})

    返回倒数第1个开始，长度为2的切片（长度不足，只返回1个元素）

    {
   _id:1,
   class:1,
   students:[
                 {name:"WangErmazi", age:18 }
           ]
}
