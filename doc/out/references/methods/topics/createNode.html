shard.createNode()

shard.createNode(<host>,<service>,<dbpath>,[config])

在分片组中创建节点。

Note:

只有在分片组启动之后，才能创建节点操作。

参数描述

参数名 参数类型 描述 是否必填
host string 指定节点的主机名。 是
service string 节点端口号。 是
dbpath string 节点路径。 是
config Json 节点配置信息，如配置日志大小，是否打开事务等，具体可参考数据库配置。 否

格式

shard.createNode()方法的定义格式有三个参数：host、service、dbpath、config，如上表所示，前三个参数的类型都是字符串型，必填；最后一个是Json对象，选填。
("<主机名>","<端口号>","<节点路径>"[,{<configParam>:value,...}])

示例

  • 在分片组shard中创建一个端口号为60000的节点，指定日志文件大小为64MB
    shard.createNode("vmsvr2-suse-x64",60000,"/opt/sequoiadb/data/60000",{logfilesz:64})
Note:

在一个分片组中能创建多个节点，但是连个节点的端口号必须相差5以上。因为系统为每个节点后台控制了5个通信接口。
