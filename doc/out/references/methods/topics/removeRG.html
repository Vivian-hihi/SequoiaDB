db.removeShard()

db.removeShard(<name>)

删除数据库中指定的分片组。

参数描述

参数名 参数类型 描述 是否必填
name string 分片组名，同一个数据库对象中，分片组名唯一。 是

格式

removeShard() 方法的定义格式只有name字段，name的值是字符串型，必填。

(<"分片组名">)
Note:
  • 分片组名必须存在。

示例

  • 删除名为“shard”的分片组 

    db.removeShard("Shard")
