SdbDate

Json 格式

{ "$date" : <YYYY-MM-DD> }

函数格式

格式 描述
SdbDate() 当前日期
SdbDate(字符串:YYYY-MM-DD) 指定日期

示例

  • 插入一个日志类型的记录
    db.foo.bar.insert({date:{$date:"2015-03-13"}});
db.foo.bar.insert({date:SdbDate("2015-03-13")});
