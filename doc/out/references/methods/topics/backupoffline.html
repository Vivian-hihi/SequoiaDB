db.backupOffline()

backupOffline([options])

备份数据库。

参数描述

    参数名    参数类型    描述                                                            是否必填

    options    Json对象     设定备份名，指定复制组，备份方式等参数     否

  参数options选项    [GroupID]  [GroupName]  [Name]  [Path]  [IsSubDir]  [Prefis]  [EnableDataDir]  [Description]  [EnsureInc]  [OverWrite]

    GroupID

     指定备份的复制组ID，缺省为所有复制组。Example: GroupID:1000 或 GroupID:[1000, 1001]

    GroupName

     指定备份的复制组名，缺省为所有复制组。Example: GroupName:"data1" 或 GroupName:["data1", "data2"]

    Name

     备份名称，缺省为“YYYY-MM-DD-HH:mm:SS”时间格式的备份名。Example: Name:"backup-2014-1-1"

    Path

     备份路径，缺省为配置参数指定的备份路径。该路径支持通配符（g%: group name, h%: host name, s%:service name）。Example: Path:"/opt/sequoiadb/backup/%g"

    IsSubDir

     上述Path参数所配置的路径是否为配置参数指定的备份路径的子目录，缺省为false。Example: GroupID:1000 或 GroupID:[1000, 1001]

    Prefix

     备份前缀名，支持通配符（g%,h%,s%），缺省为空。Example: Prefix:"%g_bk_"

    EnableDataDir

     是否开启日期子目录功能，如果开启则会自动根据当前日期创建“YYYY-MM-DD”的子目录，缺省为false。Example: EnableDataDir:false

    Description

     备份描述。Example: Description:"First backup"

    EnsureInc

     是否开启增量备份，缺省为false。Example: EnsureInc:false

    OverWrite

     存在同名备份是否覆盖，缺省为false。Example: OverWrite:false

示例

  • 对整个数据库进行全量备份

    db.backupOffline({Name:"FullBackup1"})
