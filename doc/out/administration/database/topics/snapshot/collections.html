集合快照

描述

集合快照列出当前数据库节点中所有的非临时集合（协调节点中列出用户集合），每个集合为一条记录。

标示

SDB_SNAP_COLLECTIONS

适用节点

全部

字段信息

由于数据节点与编目节点保存的集合信息不同，集合快照在协调节点与其它节点所返回的结构有所不同：

非协调节点字段信息

字段名 类型 描述
Name 字符串 集合完整名
Details.ID 整数 集合ID，范围0-4095，集合空间内唯一
Details.LogicalID 整数 集合逻辑ID
Details.Sequence 整数 序列号
Details.Indexes 整数 该集合所包含的索引数量
Details.Status 字符串 集合当前状态
  • Free：空闲
  • Normal：正常
  • Dropped：被删除
  • Offline Reorg Shadow Copy Phase：离线重组复制阶段
  • Offline Reorg Truncate Phase：离线重组清除阶段
  • Offline Reorg Copy Back Phase：离线重组重入阶段
  • Offline Reorg Rebuild Phase：离线重组重建索引阶段
NodeName 字符串 节点名（主机名+端口号）
GroupName 字符串 节点所在分区组名（仅在集群模式下显示）

协调节点字段信息

字段名 类型 描述
Name 字符串 集合完整名
Details.GroupName 字符串 节点所在分区组名
Details.Group.ID 整数 集合ID，范围0~4096，集合空间内唯一
Details.Group.LogicalID 整数 集合逻辑ID
Details.Group.Sequence 整数 序列号
Details.Group.Indexes 整数 该集合所包含的索引数量
Details.Group.Status 字符串 集合当前状态
  • Free：空闲
  • Normal：正常
  • Dropped：被删除
  • Offline Reorg Shadow Copy Phase：离线重组复制阶段
  • Offline Reorg Truncate Phase：离线重组清除阶段
  • Offline Reorg Copy Back Phase：离线重组重入阶段
  • Offline Reorg Rebuild Phase：离线重组重建索引阶段
Details.Group.NodeName 字符串 节点名（主机名+端口）

非协调节点示例

> db.snapshot(SDB_SNAP_COLLECTIONS)
{
  "Name": "foo.bar",
  "Details": [
    {
      "ID": 0,
      "LogicalID": 0,
      "Sequence": 1,
      "Indexes": 8,
      "Status": "Normal",
      "NodeName": "vmsvr2-suse-x64:11820",
      "GroupName": "datagroup1"
    }
  ]
}

协调节点示例

> coord.snapshot(SDB_SNAP_COLLECTIONS)
{
  "Name": "susefoo.susebar",
  "Details": [
    {
      "GroupName": "datagroup1",
      "Group": [
        {
          "ID": 0,
          "LogicalID": 0,
          "Sequence": 1,
          "Indexes": 1,
          "Status": "Normal",
          "NodeName": "vmsvr2-suse-x64:11820"
        }
      ]
    }
  ]
}
