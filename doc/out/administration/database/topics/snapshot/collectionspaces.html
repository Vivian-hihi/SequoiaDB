集合空间快照

描述

集合空间快照列出当前数据库节点中所有的集合空间，每个集合空间为一条记录。

标示

SDB_SNAP_COLLECTIONSPACES

适用节点

全部

字段信息

由于数据节点与编目节点保存的集合空间信息不同，集合空间快照在协调节点与其它节点所返回的结构有所不同：

非协调节点字段信息

字段名 类型 描述
Name 字符串 集合空间名
Collection 字符串数组 集合空间中所包含的所有集合
PageSize 整数 集合空间数据页大小
GroupName 字符串 集合空间所在的分区组名

协调节点字段信息

字段名 类型 描述
Name 字符串 集合空间名
Collection 字符串数组 集合空间中所包含的所有集合
PageSize 整数 集合空间数据页大小
Group.GroupID 整数数组 该集合空间所在的分区组ID列表
Group.GroupName 字符串 该集合空间所在的分区组名列表

非协调节点示例

> db.snapshot(SDB_SNAP_COLLECTIONSPACES)
{
  "Collection": [
    {
      "Name": "bar"
    }
  ],
  "PageSize": 4096,
  "Name": "foo",
  "GroupName": "datagroup1"
}

协调节点示例

>coord.snapshot(SDB_SNAP_COLLECTIONSPACES)
{
  "Collection": [
    {
      "Name": "bar"
    }
  ],
  "Group": [
    {
      "GroupID": 1000,
      "GroupName": "datagroup1"
    }
  ],
  "Name": "foo",
  "PageSize": 4096,
  "_id": {
    "$oid": "52451a6bf80be5d22b27489b"
  }
}
