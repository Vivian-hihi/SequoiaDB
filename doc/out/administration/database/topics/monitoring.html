监控

概念

监控是一种监视当前系统状态的方式。在SequoiaDB中，用户可以使用快照（SNAPSHOT）与列表（LIST）命令进行系统监控。

快照

快照是一种得到系统当前状态的命令，主要分为以下类型：

快照标示 快照类型 适用节点类型 描述
SDB_SNAP_CONTEXTS 上下文 数据，编目 上下文快照列出当前数据库节点中所有的会话所对应的上下文
SDB_SNAP_CONTEXTS_CURRENT 当前会话上下文 数据，编目 当前上下文快照列出当前数据库节点中当前会话所对应的上下文
SDB_SNAP_SESSIONS 会话 数据，编目 会话快照列出当前数据库节点中所有的会话
SDB_SNAP_SESSIONS_CURRENT 当前会话 数据，编目 当前会话快照列出当前数据库节点中当前的会话
SDB_SNAP_COLLECTIONS 集合 全部 集合快照列出当前数据库节点或集群中所有非临时集合
SDB_SNAP_COLLECTIONSPACES 集合空间 全部 集合空间快照列出当前数据库节点或集群中所有集合空间（编目集合空间除外）
SDB_SNAP_DATABASE 数据库 数据，编目 数据库快照列出当前数据库节点的数据库监视信息
SDB_SNAP_SYSTEM 系统 数据，编目 系统快照列出当前数据库节点的系统监视信息
SDB_SNAP_CATALOG 编目信息 协调 用于查看编目信息

列表

列表是一种轻量级的得到系统当前状态的命令，主要分为以下类型：

列表标示 列表类型 适用节点类型 描述
SDB_LIST_CONTEXTS 上下文 数据，编目 上下文列表列出当前数据库节点中所有的会话所对应的上下文
SDB_LIST_CONTEXTS_CURRENT 当前会话上下文 数据，编目 当前上下文列表列出当前数据库节点中当前会话所对应的上下文
SDB_LIST_SESSIONS 会话 数据，编目 会话列表列出当前数据库节点中所有的会话
SDB_LIST_SESSIONS_CURRENT 当前会话 数据，编目 当前会话列表列出当前数据库节点中当前的会话
SDB_LIST_COLLECTIONS 集合 全部 集合列表列出当前数据库节点或集群中所有非临时集合
SDB_LIST_COLLECTIONSPACES 集合空间 全部 集合空间列表列出当前数据库节点或集群中所有集合空间（编目集合空间除外）
SDB_LIST_STORAGEUNITS 存储单元 数据，编目 存储单元列表列出当前数据库节点的全部存储单元信息
SDB_LIST_GROUPS 分片组 协调 分片组列表列出当前集群中的所有分区信息
