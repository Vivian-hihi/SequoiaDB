Sequoiadb应用程序开发 / PHP驱动

数据库编译及PHP插件

Windows

假设sequoiadb的源路径为：C：\sequoiadb

  • 打开SDK,cd 到 c:\sequoiadb 的目录下,输入 
    scons
  • cd 到 c:\sequoiadb\driver\php5 的目录下,输入 
    scons --phpversion=<5.3.10> (尖括号的是PHP版本,根据运行的版本去编译相应版本的插件)
  • 把 c:\sequoiadb\driver\php5\build\normal 下的 sequoiadb.dll 拷贝到 c:\php\ext 目录下
  • 拷贝后，在php.ini添加一行 
    extension=sequoiadb.dll
  • 再次重启Apache服务
  • 验证是否编译成功

    在 C:\apache\htdocs 目录下新建一个 phpinfo.php 文件,内容如下 

    保存后,打开网页,输入网址: 127.0.0.1/phpinfo.php
            查找字符 sequoiadb 找到就证明完成。

Ubuntu Linux

假设：sequoiadb的源码路径: /home/jwhe/sequoiadb

  • cd 到 /home/jwhe/sequoiadb 的目录下,输入 
    scons
  • cd 到 /home/jwhe/sequoiadb/driver/php5 的目录下,输入 
    scons --phpversion=<5.3.10> (尖括号的是PHP版本,根据运行的版本去编译相应版本的插件)
  • 把 /home/jwhe/sequoiadb/driver/php5/build/normal 下的 libsequoiadb.so 拷贝到 /usr/lib/php5/<8位数字的文件夹> 目录下
  • 拷贝后，需要在 /etc/php5/apache2/php.ini添加一行 extension=libsequoiadb.so
  • 再次重启Apache服务 
    sudo /etc/init.d/apache2 restart
  • 在 /var/www 目录下新建一个 phpinfo.php 文件,内容如下 
    保存后,打开网页,输入网址: 127.0.0.1/phpinfo.php 查找字符
            sequoiadb 找到就证明完成。
