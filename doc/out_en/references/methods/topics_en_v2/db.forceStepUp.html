db.forceStepUp()

NAME

forceStepUp - 在一个不具备选举条件的复制组中，强制将一个备节点升级为主节点。请谨慎使用该命令!

SYNOPSIS

db.forceStepUp([options])

CATEGORY

Sequoiadb

DESCRIPTION

在一个不具备选举条件的复制组中，强制将一个备节点升级为主节点。请谨慎使用该命令!

options (json object)
参数集合
Seconds
强制升级为主节点的持续时间
Note:
  • 目前仅在catalog组中开放此功能。
  • 目标复制组中不能存在主节点，且其他节点的LSN不能比目标节点的LSN大。
  • 当持续时间到期，所有节点会重新按照选举规则进行选举。
  • 如果创建了用户，在主catalog节点的情况下无法直接连接。可以先关闭catalog的鉴权功能(auth参数)再进行操作。

Examples

连接catalog节点(host1:30000)，并使其强制升主， 持续300s。

var db = new Sdb("host1", 30000) ;
db.forceStepUp({Seconds:300});
