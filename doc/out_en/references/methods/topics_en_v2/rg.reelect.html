rg.reelect()

NAME

reelect - 在复制组中进行重新选举。

SYNOPSIS

rg.reelect([options])

CATEGORY

ReplicaGroup

DESCRIPTION

在复制组中进行重新选举。

options (json object)
参数集合
Seconds
重新选举需要在多少秒内完成
Note:
  • 返回超时错误代表在规定时间内重选没有完成。通过db.listReplicaGroups()观察最终结果。
  • 只有复制组中存在主节点时才可以进行重新选举。

Examples

在"datagroup1"中进行重新选举，超时时间为60s。

var rg = db.getRG("datagroup1") ;
rg.reelect({Seconds:60});
