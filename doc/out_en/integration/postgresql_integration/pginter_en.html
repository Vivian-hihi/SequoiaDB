Integration with PostgreSQL

PostgreSQL is a open-source relational database, it supports standard SQL and can be used to develop applications with JDBC driver connecting. By extending PostgreSQL's functions, developer can use SQL to access SequoiaDB database, can do CRUD in SequoiaDB database.
