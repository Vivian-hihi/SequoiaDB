Query Data

Name Description Example
Request Header Same as general request header  
Request Content cmd: query
name: full name of collection(CollctionSpace.Collection) space
sort: sort field name(optional)
selector: query the result column(optional)
skip: how many rows to skip(optional)
returnnum: max return records number(optional)
filter: query condition(optional)
 cmd=query&name=cs.cl&filter={"name":"hello"}
Description

errno: return value, 0 for success others means failure

}
