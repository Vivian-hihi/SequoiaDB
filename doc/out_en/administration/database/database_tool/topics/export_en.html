Data Migration — Export

sdbexprt

sdbexprt is a practical tool, it can export a JSON format or a CSV format data stored file from SequoiaDB database .

Options

Param Description
--help return the basic help and usage information.
--hostname,-h export data from Sequoiadb of the specified hostname.In default,the sdbexprt try to connect to local hostname.
--svcname,-s specify port.In default,the sdbexprt try to connect to the host which port number is 50000.
--type specify the format of export data.It can be CSV or JSON.
--file specify the name of file want to export.
--delchar,-a specify the charcater separator.The default is ","csv" formats.
--delfield,-e specify the field delimiter. "," is default, "csv" formats.
--delrecord,-r specify the record delimiter."\n" is default.
--fields specify one or more field names to export,separated by commas(,).
--csname,-c specify the collectionspace name of exported data.
--clname,-l specify the collectionspace of exported data.

Usage

In the following sample,sdbexport will export data from the local database which port is 50000,and collectionspace name is foo,collection name is bar,and export type is csv,export file name is contact,export field are field1 and field2.

sdbexprt -h localhost –s 50000 –-type csv –-file contace --fields field1,field2 –c foo –l bar
