SequoiaDB Database Description

SequoiaDB Enterprise Edition

SequoiaDB® Enterprise Edition provides scalable database server software to handle the demanding workloads of large and midsize enterprise servers. It delivers high performance across multiple workloads, including in-memory computing, ad-hoc queries and powerful integrated analytics. It also provides build-in security, automated management and visual administration tools while helping to reduce administration, storage, development and server costs. SequoiaDB® Enterprise Edition runs on Linux® platform.

SequoiaDB® Enterprise Edition features include:

  • SequoiaDB Enterprise Server
  • SequoiaDB Administration Center
  • Expert Support and Service
