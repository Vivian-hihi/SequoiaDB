SEE Analytics Engine

SequoiaDB Enterprise Edition Analytics Engine includes integration with Apache Spark. And you can optionally install and configure your own Hadoop cluster, in order to better utilize some features in Apache Spark.

SEE Analytics Engine is coming with SequoiaDB Enterprise Edition installation package. Please refer Spark Integration section for installing SEE Analytics Engine separately.
