Automatic installing

OM supports IE 7/8/9+, Chrome, Firefox and other main stream web browsers.

  • Step1: Login OM

    1.1 Access with web browser, address is the OM's address, and the default port number is 8000.

    1.2 For example, http://192.168.1.100:8000/login.html

    1.3 Login user name default is admin, psw admin.

    1.4 Input the default psw admin.

    1.5 Click <登录> button.

  • Step2: Deployment

    2.1 Click the <部署引导> in the navigation bar.

    2.2 Cluster's parameters have been filled in as default, we are not required to change too much except the <集群名> and <业务名>, because each cluster's name must be unique in database. If you want to change more parameters, you can choose the <高级选项> under. After all set, click <确定> to continue.

  • Step3: Add host

    3.1 Input the <地址>, <用户名>, <密码>, <SSH 端口> in the operation column on the left, the system will show the host already added. The <地址> not only supports IP address, and Hostname to scanning but also supports segments of IP and Hostnames. If you need help with the <地址>, you can click <帮助>.

    3.2 Input the <地址>, <用户名>, <密码>, <SSH 端口> in the operation column on the left, then click <扫描> to sacn.

    3.3 After scanning, the result will be shown on the right. Correct hosts will be tagged by default, and those not chosen are incorrect hosts, and need to re-input. You can choose the host and re-input the parameters then scan again. Click input outside the input box can start the scanning.

    3.4 In the list of hosts, you should choose the hosts you want to add, tag them, click <下一步>.

    3.5 Wait for OM to get hosts' information.

    3.6 Left is the list of hosts, detail info of the host is on the right, choose a host on the list, you can check the detail info on the right.

    3.7 In the host list, blue bubble means the number of chosen disks, yellow bubble means the number of host's warning. (When pointed, system will pump up notes)

    3.8 Ensure the hosts are all correct, click <下一步>, when completed, system will automatically jump to the next page.

  • Step 4: Create Business

    4.1 Setting business information, you can pull down the list and choose <集群模式> or <独立模式>

    4.2 <高级选项>, we can choose one or multiple hosts to install business, default arrange by the system.

    4.3 Set the parameters, click <下一步>.

    4.4 At the left, there shows the information of each cluster and business, we can use business to add partition group, and we can use the partition group to add, remove nodes. (Warning: Catalog group and Coord group can't be removed).

    4.5 At the right, there is a list of nodes, choose a row then you can edit the configuration of the host.

    4.6 To Batch edit the node, we can tag the row you want to edit, and click <选择操作> - <全选> - <反选>, <已选定操作> - <修改节点配置> to edit.

    4.7 The second row of the list , we have boxes of query conditions, input parameters, we can find the node we want.

    4.8 To Batch edit the node, <数据路径>, <服务名> can be set in batch by specific rules you give. Click <帮助> to have further instruction for the operation.

    4.9 After setting, click <下一步> to install all the hosts.

  • Step 5: Installing business

    5.1 Start installing business.

    5.2 Installing completed, click <返回>.

    5.3 Business installing completed.

  • 步骤六：主页左边是集群列表，右边是集群实时监控信息

  • 步骤七：创建集群

    7.1 点击 <状态> 旁边的下拉菜单；

    7.2 点击 <创建集群>；

    7.3 输入集群参数，点击 <高级选项> 可以设置更多参数；

    7.4 点击 <确定>，完成创建。

  • 步骤八：查看业务信息和删除业务

    8.1 查看当前已安装的业务（图中 myBusiness 是已经安装的一个业务，myBusiness 是业务名）；

    8.2 点击表格 <业务> 查看业务信息（一）；

    8.3 点击业务名旁边的下拉菜单，点击 <业务列表> 看业务信息（二）；

    8.4 删除业务，在业务列表，点击业务对应的 <删除业务> 按钮，弹出警告窗口，点击 <确定> 开始删除业务，等待卸载完成，完成后弹出提示，点击 <返回>。

  • 步骤九：查看集群主机数量和主机详细信息，删除主机

    9.1 点击要查看的集群名；

    9.2 鼠标移动到主机旁边蓝色的徽章，徽章上面的数字就是当前主机数量；

    9.3 表格第二列是显示当前状态信息，黄色表示 warning，红色表示 danger；

    9.4 主页右边的 <实时监控>，监控集群的平均 CPU 使用率， 平均内存使用率， 平均磁盘使用率，网络入口流量，网络出口流量，网络每秒接收数据包个数，网络每秒发送数据包个数；

    9.5 点击表格 <主机>，查看主机列表；

    9.6 点击业务名旁边的下拉菜单，点击 <主机列表>查看主机列表；

    9.7 删除主机，选择要删除的主机，点击 <已选定操作>，点击 <删除主机>，弹出警告窗口，点击 <确定> 开始删除主机，等待完成，完成后弹出删除结果，点击 <关闭>。

  • 步骤十：删除集群

    10.1 回到主页，点击要删除的集群旁边下拉菜单；

    10.2 点击菜单的 <删除集群>；

    10.3 删除完成。
