sdbdpsdump是一个 SequoiaDB 数据库的同步日志文件dump工具。它可以解析replica log同步日志文件的内容，并且给出结果报告。

##权限需求##

运行 sdbdpsdump 命令的用户必须对数据库的数据与索引文件拥有读权限。

##连接需求##

sdbdpsdump 不需要与数据库连接。

##选项##
+-------------+------+-----------------------------------------------------------------------------------+
| 参数        | 参数 | 描述                                                                              |
+=============+======+===================================================================================+
| --help      | -h   | 返回基本帮助和用法文本                                                            |
+-------------+------+-----------------------------------------------------------------------------------+
| --version   | -v   | 返回工具的编译版本                                                                |
+-------------+------+-----------------------------------------------------------------------------------+
| --meta      | -m   | 指定在解析日志文件时，也解析日志文件的元数据信息，不可单独使用                    |
+-------------+------+-----------------------------------------------------------------------------------+
| --type      | -t   | 指定只解析日志文件中指定类型的日志：                                              |
|             |      | 0：         空白填充                                                              |
|             |      | 1：         数据插入                                                              |
|             |      | 2：         数据更新                                                              |
|             |      | 3：         数据删除                                                              |
|             |      | 4：         创建集合空间                                                          |
|             |      | 5：         删除集合空间                                                          |
|             |      | 6：         创建集合                                                              |
|             |      | 7：         删除集合                                                              |
|             |      | 8：         创建索引                                                              |
|             |      | 9：         删除索引                                                              |
|             |      | …etc 更多类型，请查看：dpsDef.hpp 中的 DPS_LOG_TYPE 类型枚举                      |
+-------------+------+-----------------------------------------------------------------------------------+
| --name      | -n   | 指定集合空间或者集合名，只解析和指定名相关的集合或者集合空间的日志                |
+-------------+------+-----------------------------------------------------------------------------------+
| --lsn       | -l   | 指定lsn值，只解析lsn的日志，可与-a和-b联用                                        |
+-------------+------+-----------------------------------------------------------------------------------+
| --last      | -e   | 指定解析日志文件最后的N条日志，N为指定的数值（需要-s指定到文件）                  |
+-------------+------+-----------------------------------------------------------------------------------+
| --source    | -s   | 指定日志文件的目录或路径，默认为当前目录                                                |
+-------------+------+-----------------------------------------------------------------------------------+
| --output    | -o   | 指定输出文件，默认为屏幕输出                                                      |
+-------------+------+-----------------------------------------------------------------------------------+
| --ahead     | -a   | 指定解析指定lsn之前的N条日志，N为指定数值，必须与-l/--lsn联用，默认为20           |
+-------------+------+-----------------------------------------------------------------------------------+
| --back      | -b   | 指定解析指定lsn之后的N条日志，N为指定数值，必须与-l/--lsn联用，默认为20           |
+-------------+------+-----------------------------------------------------------------------------------+

##用法##


在下面的例子，sdbdpsdump 解析指定目录中的replica log文件，并只解析类型为1（数据插入）的日志，输出到当前目录out.log。

<pre class="prettyprint lang-javascript">
> sdbdpsdump -s /data/database/40000/replicalog/sequoiadbLog.0 -o out.log -t 1</pre>
